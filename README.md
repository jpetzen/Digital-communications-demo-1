## Installation guide

```sh
# clone the project
git clone https://github.com/jurerotar/Digital-communications-demo.git

# Project setup
npm install

# Compiles and hot-reloads for development
npm run serve

# Compiles and minifies for production
npm run build

# Lints and fixes files
npm run lint
```
